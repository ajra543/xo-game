# Neon Tic-Tac-Toe (XO) Web Application

A visually stunning, modern Web-based 3x3 Tic-Tac-Toe game. It uses a Flask (Python) backend to handle server requests and process the computer's moves using a rule-based AI, coupled with a responsive, glassmorphic HTML/CSS/JS frontend.

## Features

- **Beautiful Dark Mode & UI**: Premium glassmorphic interface with interactive neon glows, custom Google Fonts, and smooth hover/click scales.
- **Two Game Modes**:
  1. **2 Player (Local)**: Take turns locally sharing the mouse.
  2. **VS Computer (AI)**: Face a smart rule-based AI processed on the Flask backend.
- **Scoreboard**: Tracks Player X wins, Player O (or Computer) wins, and Draws.
- **Instant Reset**: Start new matches instantly while keeping your scoreboard history.

## How to Run

1. Open your terminal in this directory.
2. Run the Flask server:
   ```bash
   .venv/Scripts/python app.py
   ```
3. Open your browser and navigate to:
   ```text
   http://127.0.0.1:5000/
   ```
4. Start playing!
